SM-100は、USB PD 100Wで動作するオープンソースの小型3Dプリンターです。

## 背景

東京の狭小賃貸物件に小型犬と一緒に暮らしていると、3Dプリンターの置き場所に悩まされます。
2024年に勢いで初めて購入したBambu Lab A1 Miniという3Dプリンターは、安価でバランスのいい製品でしたが、ベッドが前後に動く方式（ベッドスリンガー）であることなどから結構設置面積が必要になります。
またエンクロージャーがないので犬が届く位置に設置すると怪我や火傷のリスクがあり、プラスチックを溶かすことによって発生するVOCの人体や小型犬への影響も心配です。

近年の市販3Dプリンターはツールチェンジャーの搭載などにより大型化の傾向にあり、私のニーズを満たす小型3Dプリンターはおそらく登場しそうになかったので（出たとしても実際そこまでの台数が売れない気がします）、小型3Dプリンターの製作を開始しました。[Fraxinus 00w2](https://fraxinus.jp/en/docs/micro-printers/#fraxinus00w2)や[Fraxinus 00cw](https://fraxinus.jp/en/docs/community/#fraxinus00cw)、[Fraxinus 00tcw](https://x.com/c_bata_/status/1891012343326261727)、そして [Voron 0.2](https://github.com/voronDesign/Voron-0) など複数の小型3Dプリンターを製作しましたが、どれも自分が今後日常的に使い続けるマシンとして考えると気になる点が残っていました。そこでの知見を元に設計したのがSM-100です。特に次の3点を重視しています。

1. **Compact Footprint**: 私の場合、3Dプリンターを使う頻度には波があり、よく使う月もあれば、ほとんど使わない月もあります。忙しい時期には数週間触らないこともあり、そういうときに大きなスペースを占有するマシンだと、家族からも不満を言われいつか手放さざるを得なくなることは明白でした。場所を取らないサイズであることは3Dプリンターを長く持ち続けるうえでとても重要です。
2. **Thermal Management**: ステッピングモーターの発熱が大きすぎると、トルク低下や寿命の低下につながります。以前製作したVoron 0.2では、LDO製のNEMA 14モーターを採用し、 `run_current` も調整していましたが、トルク低下による脱調がたまに起きていました。ファンによる冷却は驚くほど効果がありこの問題を完全に解消できますが、モーターそれぞれにファンを付けてしまうと部品数や配線の手間が増えます。そこで SM-100 では、A/B/Z モーターと制御ボード類を本体下部にすべてまとめ、前面に設置した2つの4010ファンでまとめて冷却できる構成にしました。
3. **Fire Safety**: 市販品も含めて3Dプリンターは火災に注意が必要です。以前私が製作してきたFraxinus 00cwや00tcwは、ABSなどのプリント部品でエンクロージャーを印刷していましたが、一度火がつくと燃え広がってしまうことから日常的に使う気持ちにはなれませんでした。SM-100はステンレス板金部品を使用し、仮にツールヘッドなどから火が上がっても延焼リスクを抑えることができます。

SM-100はようやく自分が目指していた理想の1台になった気がします。Voron 0.2とのサイズ比較画像を下に貼りますがデスクにおいてもじゃまにならないサイズで、印刷品質も十分実用的です (PRUSAやBambu Labの3Dプリンターにはおよびません)。長く使う上では構造がシンプルで壊れづらく、仮に壊れても修理しやすいこともまた重要ですが、その点もいろいろと考慮して設計することができ、長く使い続けられそうなプリンターになりました。

## 特徴

**Lower-Mounted Motion System** — A/Bモーターを下部にまとめ、5mmのステンレスシャフトで中継しています。このおかげで重心が下がり、上部チャンバーに余分な気流を作らなくても、前面吸気ファンでモーターや電子部品を同時に冷却できます。

**Bowden Extruder and Side-Mounted Cooling** — ボーデンエクストルーダー方式を採用し、パーツ冷却ファンはツールヘッドではなく側面に設置することで小型化を測ります。

**Sheet-Metal Chassis** — Voron V0 のようなアルミフレーム構成と比べても、板金はより薄い断面で同等レベルの剛性を確保しやすく、SM-100 のコンパクトさに貢献しています。また、プリント製の外装部品よりも防火性に優れます。

**MGN9C Linear Rails** - リニアレールの価格は長さによって大きく変わりますが、SM-100で使っている100mmと150mmのレールはAliExpressで比較的安価に入手でき、ガタツキの問題もそれほどありません。またVoron 0.2ではMGN7Hが採用されていますが、M2ネジが緩みやすいことや安価なリニアレールを何個か購入してきた経験上ガタツキがMGN9Cより大きい傾向にあるように思います。MGN7Hは摩耗によるガタツキの影響も出やすいと聞いたことがありSM-100ではMGN9Cを採用しています。

**Support for NEMA14 and NEMA17 Stepper Motors** - 当初は小型化のためにVoron 0.2と同じくNEMA 14ステッピングモーターを前提に設計していましたが、より安価に入手しやすいNEMA 17にも対応しました。私が製作したSM-100も、メルカリでQIDI Tech社の安価なジャンク品を購入し、分解して取り出したNEMA 17モーターを使っています（私はspeed boat raceとかをしたいわけではないので、安価なNEMA 17モーターで困ったこともありません）。

**USB PD Power Delivery** — 100W USB-PD のみで動作するため、大型の内蔵電源が不要になり、全体の設置サイズをさらに抑えられます。
